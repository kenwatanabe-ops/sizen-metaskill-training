PART 03 心 — Mental / Leadership

心を鍛える

リーダーシップとは、技術ではなく世界観。エネルギーとフォーカスをどこに向けるか — それが「心」の本質。

3-1. リーダーシップとは何か (定義)

リーダーシップは「技術」ではなく「世界観」である。
リーダーが見ている方向に、チームは進んでいく。 — Dr.遠藤先生
公式

リーダーシップ = エネルギー × フォーカス

エネルギー = ステート (心身の状態) を高く保つ力。エネルギーが高く、柔軟性のあるものが環境を支配する。
フォーカス = 向かうべきゴールに意識を集中させる力。意識が向いたものが現実化する。

📌 苫米地理論との接続 — エフィカシー

エネルギー = エフィカシー (Self-Efficacy) = ゴール達成能力の自己評価。
エフィカシーが高い = 「高難度ゴールでも自分なら届く」と確信できる状態。リーダーの役割は メンバーのエフィカシーを上げること。「あなたはできる」を100%信念で言い切る = エフィカシー注入。

3-2. 影響力 — 世界観を売る技術

世の中はすべてセールスである

ネガティブな世界観 (買わない)リーダーが売り返す世界観
「自分はできない」「これは無理だ」「あなたはできる」「やり方を一緒に考えよう」
「メンバーは仕事ができない」前提「どれくらい仕事ができるか」前提

100%の信念 vs 99%の信念

天と地の差

「99%できる」と「100%できる」と信じている間には、天と地ほどの差がある。人は言葉そのものより、話し手の 「確信度」 と 「エネルギー」 を感じ取る。

📌 苫米地理論 — ゴール = 現状の外 / want to vs have to

📌 苫米地理論 — 抽象度を上げる = IQを上げる

苫米地式IQの定義 = 抽象度を自在に上下できる能力。一般的なIQテストとは別概念。

  • 抽象度を上げる = 多様な情報を統合して構造として捉える
  • 抽象度が高い = 「もっと大きな問い」を立てられる
  • 同じ事象を別の抽象度で見ると新しい解釈が生まれる
  • リーダー・経営者は より高い抽象度 で意思決定する

→ チャンクアップ訓練 = IQ向上訓練。「目の前の問題」を1段上げると「業界の問題」、もう1段で「社会の問題」、もう1段で「人類の問題」。

📌 苫米地理論 — RAS (網様体賦活系)

これら3フィルターの 科学的実装 = RAS (Reticular Activating System)。脳幹にある情報フィルタリング機構で、毎秒数百万bitの情報のうち、どれを意識に上げるかを決定する。
ゴール設定により RAS のフィルタリング基準が変わる = 99.999%中の何を拾うかが書き換わる。「赤い車が欲しい」と思った瞬間から街中の赤い車が目につく現象の正体。
